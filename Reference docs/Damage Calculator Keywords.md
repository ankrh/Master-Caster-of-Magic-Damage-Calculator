# Damage Calculator — Relevant Keywords

Reference list for filtering forum posts, patch notes, and manual excerpts.
A post is **relevant** if it mentions any keyword in the "Relevant" section
below. A post discussing only the "Not Relevant" categories can be skipped.

Match case-insensitively. Treat singular/plural and obvious variants
(e.g. "tohit" / "to-hit" / "to hit") as equivalent. Hyphens, spaces, and
slashes between words are interchangeable.

## Not Relevant (skip-list)

A post is safe to skip if it *only* discusses these topics (and no keyword
from the Relevant list appears):

- Economy: gold, mana, food, production, income, upkeep, taxes, tax rate
- City management: buildings, population, farmers, workers, rebels,
  happiness, housing, granary, marketplace
- Research: research, spellbook, spellbooks, spell rarity, common/uncommon/
  rare/very rare (as rarity tiers, not outcomes)
- Overland spells not touching combat stats: Earth Lore, Enchant Road,
  Move Fortress, Awareness, Nature's Awareness, Detect Magic, Spell of
  Mastery, Spell of Return, global summoning cost
- Hero/item forging: item creation, artifact forging, hero recruitment
  cost, hiring fee, item powers (only skip if not a combat stat bonus)
- Movement: movement points, flying (movement only), walking, swimming,
  sailing, roads, terrain movement cost, teleport, Wind Mastery
- Diplomacy: wizard pact, alliance, treaty, declare war, trade spells,
  exchange, tribute
- Map / exploration: towers, lairs, nodes (as exploration targets, not
  auras), rumors, rare encounters, treasure
- AI / difficulty: AI behavior, difficulty level, Intro/Easy/Normal/Hard/
  Impossible
- Interface / technical: hotkeys, UI, graphics, sounds, music, save file,
  save/load, crash, memory leak, installer, Steam, GOG, multiplayer,
  hotseat, LAN
- Banishment: banish, banished, return-home (only skip if not triggering
  combat)
- Flavor text: lore, story, narration, wizard portrait, unit artwork
